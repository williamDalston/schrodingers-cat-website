# Article Title

> **Writing Guide**: This is a template for creating new articles. Copy this file and rename it to match your article ID (e.g., `qc-primer-004.md`).

## Article Structure

### Opening Hook
Start with a compelling opening that grabs the reader's attention. Use:
- A thought-provoking question
- A surprising fact
- A relatable scenario
- A quote from the McQueen interview

### Introduction
- Set up the problem or question
- Introduce key concepts
- Preview what you'll cover

### Main Content Sections
Use H2 headings for major sections. Each section should:
- Have a clear focus
- Flow logically to the next
- Include examples or analogies
- Reference transcript hooks where relevant

### Quotes from McQueen Interview
When quoting from the interview, use blockquotes:

> "Direct quote from Kelvin McQueen here"
> — Kelvin McQueen, Quantum Consciousness Interview

### Examples and Analogies
- Use concrete examples
- Draw analogies to familiar concepts
- Include visual descriptions (remember to add actual images in the article folder)

### Technical Content
For complex concepts:
- Start with intuition
- Build up to technical details
- Use code blocks for formulas or technical notation

### Visual Descriptions
When referencing visuals:
- Describe what the image/diagram shows
- Explain its significance
- Add images to `/public/articles/[slug]/` folder

### Conclusion
- Summarize key points
- Connect back to the opening
- Suggest next steps or related articles

## Markdown Tips

### Headings
- Use `#` for H1 (title)
- Use `##` for H2 (major sections)
- Use `###` for H3 (subsections)

### Emphasis
- **Bold** for key terms
- *Italic* for emphasis
- `Code` for technical terms

### Lists
- Use bullet points for unordered lists
- Use numbered lists for sequential steps

### Links
- `[link text](url)` for internal/external links
- Link to related paradoxes: `/paradoxes/schrodingers-cat`
- Link to related articles: `/articles/[slug]`

### Images
- Store in `/public/articles/[slug]/` folder
- Use: `![Alt text](/articles/[slug]/image-name.jpg)`
- Always include descriptive alt text

### Code Blocks
- Use triple backticks for code blocks
- Specify language for syntax highlighting: ```javascript
- Use inline code with single backticks: `term`

## SEO Best Practices

1. **Use target keywords naturally** (from article.seoKeywords)
2. **Include internal links** to related paradoxes and articles
3. **Add descriptive alt text** to all images
4. **Use descriptive headings** (H2/H3 should contain keywords where natural)
5. **Write meta-worthy first paragraph** (used in search snippets)
